自前AI: JSポリシー（decide(state) を編集）

state は ai-interface.md の部分観測ビュー。CP/行動数/隣接性/霧はサーバー側(Engine.validateActions)が強制するため、無効な行動は自動的に無視されます。

自前AI: ローカルLLM 接続（L3 / OpenAI互換）

ブラウザから直接呼ぶため、LLMサーバ側で CORS許可 が必要です（例: Ollama/LM Studio/vLLM の OpenAI互換API）。 deadline(1000ms)超過や失敗時はそのtickを idle 扱いにします。
